## Migration Note

The old English examples in this file used an older ordering that placed explanation before action.
The current preferred ordering is:

1. Risk Level
2. Risk Type
3. Install Recommendation
4. Security Advice
5. Reasoning
6. Detailed evidence

## Usage Guidance

- If the user asks in English, follow `examples-en.md`
- If the user asks in Chinese, follow `examples-zh.md`
- If maintaining older reports, update them toward the new ordering instead of extending the old format here
